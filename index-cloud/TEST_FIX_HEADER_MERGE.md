# 表头智能合并 - 测试数据修复说明

## 问题原因

之前的测试数据使用了**空字符串 `""`** 来实现合并效果，但新的智能合并功能是**通过检测重复值来自动合并**，空字符串会被跳过而不创建单元格，导致出现空白。

## 修复内容

### 修复前（❌ 错误用法）

```java
List<List<String>> complexHeaders = Arrays.asList(
    Arrays.asList("学号", "成绩信息", "成绩信息", "成绩信息", "成绩信息"),
    Arrays.asList("", "语文", "语文", "数学", "数学"),      // ❌ 使用 ""
    Arrays.asList("", "平时", "期末", "平时", "期末")       // ❌ 使用 ""
);
```

**问题：**
- 第 2、3 行的第一列是空字符串 `""`
- 系统会跳过空字符串，不创建单元格
- 导致第一列第二行开始是空白的

### 修复后（✅ 正确用法）

```java
List<List<String>> complexHeaders = Arrays.asList(
    Arrays.asList("学号", "成绩信息", "成绩信息", "成绩信息", "成绩信息"),
    Arrays.asList("学号", "语文", "语文", "数学", "数学"),      // ✅ 使用相同值
    Arrays.asList("学号", "平时", "期末", "平时", "期末")       // ✅ 使用相同值
);
```

**效果：**
- 第 2、3 行的第一列都是"学号"
- 系统会自动检测并合并这 3 个"学号"单元格
- 形成跨 3 行的合并区域

## 核心原则

### 1. 使用重复值，不要用空字符串

```java
// ❌ 错误
Arrays.asList("", "A", "B")

// ✅ 正确
Arrays.asList("父级", "A", "B")
```

### 2. 需要跨几行就写几遍相同的值

```java
// 要让"学号"跨 3 行合并
Arrays.asList("学号", "成绩信息", "成绩信息")  // 第 1 行
Arrays.asList("学号", "语文", "数学")          // 第 2 行
Arrays.asList("学号", "平时", "期末")          // 第 3 行
// 系统会自动合并这 3 个"学号"
```

### 3. 需要跨几列就写几遍相同的值

```java
// 要让"成绩信息"跨 5 列合并
Arrays.asList("学号", "成绩信息", "成绩信息", "成绩信息", "成绩信息", "成绩信息")
Arrays.asList("学号", "语文", "语文", "数学", "数学", "英语")
```

## 记忆要点

### ✅三要

1. **要使用相同的值** - 需要合并的单元格都写相同的文本
2. **要保持连续** - 相同的值必须在水平或垂直方向连续
3. **要写满所有行** - 需要跨几行就写几遍

### ❌三不要

1. **不要用空字符串** - `""` 会被跳过，不会创建单元格
2. **不要间断** - 相同的值中间不能插入不同的值
3. **不要手动控制合并** - 相信系统的自动检测能力

## 常见问题

### Q1: 如果我就是想要空白单元格怎么办？

A: 可以使用空格 `" "` 或其他占位符，但不会被合并。

### Q2: 可以部分留白吗？

A: 可以，但留白的位置不会有单元格，也不会参与合并。

### Q3: 如何验证合并效果？

A: 运行测试后打开 Excel 文件，点击单元格查看合并区域。

## 总结

记住这个核心转变：

- **旧思维**：用 `""` 表示"这里要与上面合并"
- **新思维**：用**相同的值**表示"这些要合并"

现在系统会自动检测并合并相同的值，你只需要关注表头的语义即可！🎉
