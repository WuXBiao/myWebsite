# 复杂表头正确写法指南

## 核心原则

### ✅ 三要

1. **要使用相同的值** - 需要合并的单元格必须写完全相同的文本
2. **要保持连续** - 相同的值必须在水平或垂直方向连续
3. **要写满所有行** - 需要跨几行就写几遍相同的值

### ❌ 三不要

1. **不要用空字符串** - `""` 会被跳过，不会创建单元格
2. **不要间断** - 相同的值中间不能插入不同的值
3. **不要手动控制** - 相信系统的自动检测能力

```
┌───────────────┬───────────────┐
```

### 场景 3：三层表头（跨行 + 跨列合并）

```java
List<List<String>> headers = Arrays.asList(
    // 第 1 行："学号"跨 3 行，"成绩信息"跨 6 列
    Arrays.asList("学号", "成绩信息", "成绩信息", "成绩信息", "成绩信息", "成绩信息", "成绩信息"),
    // 第 2 行："学号"继续（实现跨 3 行），科目各跨 2 列
    Arrays.asList("学号", "语文", "语文", "数学", "数学", "英语", "英语"),
    // 第 3 行："学号"继续（实现跨 3 行），具体指标
    Arrays.asList("学号", "平时", "期末", "平时", "期末", "平时", "期末")
);
```

**效果：**
```
┌──────┬────────────────────────┐
│ 学号 │        成绩信息         │
│      ├────────┬───────┬───────┤
│      │  语文   │  数学  │ 英语  │
│      ├────┬───┼───┬───┼───┬───┤
│      │平时│期末│平时│期末│平时│期末│
└──────┴────┴───┴───┴───┴───┴───┘
```

**关键点：**
- ✅ "学号"在每一行都出现（共 3 次）→ 跨 3 行合并
- ✅ "成绩信息"在第 1 行出现 6 次 → 跨 6 列合并
- ✅ "语文"在第 2 行出现 2 次 → 跨 2 列合并
- ❌ 第 2、3 行第 1 列不要用 `""`

## 常见错误对照表

| 错误写法 ❌ | 正确写法 ✅ | 说明 |
|-----------|-----------|------|
| `["", "A", "B"]` | `["父级", "A", "B"]` | 不要用空字符串 |
| `["分类"]` (1 列) | `["分类", "分类"]` (2 列) | 跨几列写几遍 |
| 第 1 行`["学号"]`<br>第 2 行`[""]` | 第 1 行`["学号"]`<br>第 2 行`["学号"]` | 跨行要重复写 |
| `["A", "B", "C"]`<br>`["", "D", "E"]` | `["A", "B", "C"]`<br>`["A", "D", "E"]` | 每行都要写完整 |

## 快速检查清单

在提交代码前，检查表头数据：

- [ ] 所有需要合并的单元格都写了相同的值
- [ ] 没有使用空字符串 `""`
- [ ] 跨 N 行就写了 N 遍相同的值
- [ ] 跨 N 列就写了 N 遍相同的值
- [ ] 相同的值是连续的，中间没有插入其他值
- [ ] 注释清楚说明了合并逻辑

## 调试技巧

如果合并效果不对：

1. **打印表头数据**
   ```java
   System.out.println("表头结构：");
   for (int i = 0; i < headers.size(); i++) {
       System.out.println("第" + (i+1) + "行：" + headers.get(i));
   }
   ```

2. **检查是否有空字符串**
   ```java
   // 查找空字符串
   for (List<String> row : headers) {
       if (row.contains("")) {
           System.out.println("发现空字符串！");
       }
   }
   ```

3. **验证重复值**
   ```java
   // 检查"学号"出现了几次
   int count = 0;
   for (List<String> row : headers) {
       for (String cell : row) {
           if ("学号".equals(cell)) count++;
       }
   }
   System.out.println("'学号'出现了 " + count + " 次，应该跨" + count + "行");
   ```

## 总结

记住这个公式：

**合并单元格数量 = 相同值出现的次数**

- 跨 2 行 = 写 2 遍相同的值
- 跨 3 列 = 写 3 遍相同的值
- 跨 2 行 3 列 = 在 2 行×3 列的范围内都写相同的值

现在你只需要关注表头的语义，合并的事情交给工具类自动处理！🎉
